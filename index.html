    Settings & accuracy

    For exact pin accuracy and better suggestions (especially in places OpenStreetMap doesn't cover well), you can plug in your own Google Maps API key. It's stored only in your browser — never sent anywhere except to Google.

    No key set — using OpenStreetMap.

    How to get a Google Maps API key

    1. Go to Google Maps Platform and create a project (or pick one you have).
    2. Enable both Places API (New) and Maps JavaScript API in the API library.
    3. Under Credentials, create an API key.
    4. Click the key → Application restrictions → Websites → add your site (https://yourname.github.io/*). This locks the key to your domain so no one else can use it.
    5. Paste the key above and hit Save key.
    Google gives a generous monthly credit that typically covers personal use for free. Autocomplete uses session tokens so a full search-and-pick counts as one billed session, not one per keystroke.
